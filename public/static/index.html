Research Abstract

Modernizing decades-old systems without rewriting a single line.

Many critical institutions still depend on Visual Basic 6 and Windows XP-era software with no source code, vendor support, or documentation. This research introduces a four-layer agentic AI middleware that observes, understands, and orchestrates these legacy applications — exposing them through modern LLM-generated interfaces while leaving the original binary completely untouched.

The Four-Layer Architecture

Each layer is independently agentic, observable and replaceable.

LLM Frontend Generation

GPT-4V driven UI synthesis converts legacy screens into modern, accessible interfaces.

SOACRS Multi-Agent Planner

Self-optimising planner using contextual bandits to route tasks across specialised agents.

Stateful DAG Orchestrator

Deterministic execution graph with rollback, retries and full traceability.

CV-RPA Bot Layer

OpenCV + Tesseract OCR + PyAutoGUI/AHK driving zero-intrusion legacy automation.
